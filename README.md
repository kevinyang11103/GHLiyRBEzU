# 家政服务管理平台系统介绍 java91

本项目是一个基于前后端分离模式的家政服务管理平台，使用 SpringBoot 和 Vue.js 等技术栈构建。以下是项目的基本概述和功能模块介绍。

## 技术栈

- **前端技术**：Vue.js, Ajax, JSON
- **后端技术**：SpringBoot, MyBatisPlus

## 功能模块

### 系统角色

#### 管理员

- **首页管理**：管理和更新首页内容。
- **个人中心**：查看和管理个人信息。
- **用户管理**：管理用户信息。
- **服务人员管理**：管理服务人员资料和工作情况。
- **服务信息管理**：发布、更新、删除服务信息。
- **服务类型管理**：管理服务分类信息。
- **服务预约管理**：处理服务预约请求。
- **服务取消管理**：处理服务取消请求。
- **服务分配管理**：分配服务任务。
- **服务进度管理**：跟踪管理服务任务进度。
- **评价信息管理**：管理服务评价信息。
- **留言反馈管理**：管理用户留言和反馈。
- **系统管理**：进行系统设置。

#### 用户

- **首页访问**：浏览首页内容。
- **服务信息浏览**：查看服务详细信息。
- **服务预约**：预约服务。
- **服务取消**：取消预约服务。
- **服务进度查询**：查看服务进度。
- **评价信息提交**：对服务进行评价。
- **留言反馈**：提交留言反馈。
- **个人中心**：管理个人信息。
- **我的收藏管理**：管理收藏服务信息。

#### 服务人员

- **个人中心**：管理个人信息和服务任务。
- **服务分配管理**：查看分配的服务任务。
- **服务进度管理**：更新管理负责的服务任务进度。
- **评价信息管理**：查看用户评价。
